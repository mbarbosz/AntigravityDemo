# Antigravity CLI - Live Demo (Master Branch)

Este repositório contém os arquivos necessários para a apresentação e execução da demonstração prática do **Antigravity CLI**.

## Conteúdo desta Branch (`master`):
- **`demo.html`**: Página principal interativa para a apresentação/demonstração ao vivo.
- **`demo_guide.md`**: Guia passo a passo com o roteiro completo, prompts e orientações para a demonstração da IA.
- **Logos & Imagens**: Recursos visuais utilizados na página da demonstração.
